Get blueprints :
curl -H "X-Requested-By: ambari" -X GET -u admin:admin http://192.168.33.17:8080/api/v1/blueprints

Get specific blueprint :
curl -H "X-Requested-By: ambari" -X GET -u admin:admin http://192.168.33.17:8080/api/v1/clusters/ambari?format=blueprint

Get registered hosts :
curl -H "X-Requested-By: ambari" -X GET -u admin:admin http://192.168.33.17:8080/api/v1/hosts

Generate / Restore cluster :
curl -H "X-Requested-By: ambari" -X POST -u admin:admin -d '@blueprint/blueprint.json' http://192.168.33.17:8080/api/v1/blueprints/ambari
curl -H "X-Requested-By: ambari" -X POST -u admin:admin -d '@blueprint/clustertemplate.json' http://192.168.33.17:8080/api/v1/clusters/ambari
